# kotlin-minesweeper

# 1단계 사전 정리  
[x] 한 칸(Square)를 추상화, 상태 값으로 갖는 것 보단 메서드를 통해 내부 객체로 표현
[x] 마인과 논 마인으로 객체 구별
[x] 랜덤 요소 (인터페이스의 주입을 통해 테스트를 최대한 편하게), 입력 받은 지뢰 갯수와 동일하면 나머진 논 마인으로 구별
    [x] 지뢰를 미리 생성 후 하나씩 큐에서 제거하는 방법도 괜찮을듯
    [x] 전체를 생성 후 셔플로 객체의 순서를 랜덤으로 변경 

# 피드백
